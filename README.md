# BarcodeApp

## Capabilities
- this app can read, share, and generate QRCodes with custom images using Carthage

## Utilizes
- MVVM architecture
- UIImagePicker
- UIActivityViewController
- Carthage: https://github.com/EFPrefix/EFQRCode
